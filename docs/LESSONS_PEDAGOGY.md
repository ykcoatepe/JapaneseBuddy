# Lessons Pedagogy — Can-do → Activity → Check

## A1-B1 scope
- A1: greetings, classroom phrases, family/people, where you live, prices, time/date, food ordering, directions…
- A2: appointments, returns/exchanges, buying tickets, clinic visits, renting, invitations…
- B1: invitations/plans, workplace chat, travel issues, advice, reservations, opinions, follow-up requests…

Current indexed path: 22 lessons total (A1: 8, A2: 6, B1: 8).

## Flow per lesson
1) **Objective (Can-do banner)**  
2) **Activities:** Shadow (segmented TTS) → Listening MCQ → Reading MCQ  
3) **Check:** self-rating ★☆☆/★★☆/★★★ (stored in progress)  
4) **Kanji Words (optional):** per-lesson mini set; type reading in kana; correct → SRS `.vocab` seed

We use the Can-do structure and keep *all lesson text/audio original*.
